##Laravel + Grunt + Compass + Foundation 5 + Bower
Configuring laravel framework with foundation 5, compass, bower components and grunt.

####Requirements
- Ruby 1.9.1+
- NodeJS
- Grunt
- Bower
- Foundation

####Installation Steps
- npm install
- bower install
- grunt
